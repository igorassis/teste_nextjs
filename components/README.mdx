# Fretebras Design System

This project is a library of web components made in React for Fretebras products.

## Dependencies
- Node.js v10+
- Yarn v1.16+

## Usage
1. Clone this project.
2. Run `cd poc-central-next`.
3. Run `yarn` to install development dependencies.
5. Run `yarn docz:dev` to see the documentation.
